## Verbconjukation im Präsens

| Singular | Plural |
| -------- | ------ |
| ich + Verbstamm + e | wir + Verbstamm + en |
| du + Verbstamm + st | ihr + Verbstamm + t |
| er/sie/es + Verbstamm + t | sie + Verbstamm + en |
| Sie + Verbstamm + en | Sie + Verbstamm + en |
